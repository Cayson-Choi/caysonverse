# 시니어 — "디지털·AI와 친해지기 + AI 안전"

## 수업 목표
- **지식**: AI가 무엇이고 일상 어디에 이미 들어와 있는지(검색·내비·스피커) 안다
- **체험**: 3D 공간 입장, AI와의 대화, AI 사진 만들기를 직접 성공해 본다 — "나도 된다"는 효능감이 핵심 목표
- **태도**: AI를 두려워하지 않되, AI 악용(보이스피싱·딥페이크)을 구별하는 경계심을 갖춘다

## 대상 특성과 배려 (이 수업의 절반은 배려 설계)
- **가입 없는 즉시 입장이 결정적**: 아이디/비밀번호 만들기에서 대부분 탈락하는데, 이 사이트는 닉네임만 치면 들어온다. 이 지점을 수업 도입에서 직접 언급해 안심시킨다
- **조작은 단순 경로만**: 이동은 "바닥 클릭"(가장 쉬움) 하나만 먼저 가르친다. 잘못 걸어가도 Esc 한 번이면 멈춘다는 것, 실수해도 아무 일도 안 생긴다는 것을 반복 안내
- **TTS(채팅 낭독)가 효자 기능**: 근처 사람의 채팅을 음성(한국어)으로 읽어 주므로 작은 글씨 부담이 줄어든다. 🔊/🔇 토글 위치를 알려준다
- 진행 인력: 수강생 5명당 보조 강사 1명 권장. 진도는 계획의 절반이 나가도 성공
- 기기: 글씨가 큰 태블릿보다 조작이 확실한 PC+마우스가 오히려 낫다 (클릭 이동)

## 타임테이블 (90분 — 쉬는 시간 포함)

| 시간 | 단계 | 내용 |
|---|---|---|
| 0–20분 | 접속 온보딩 (충분히) | 접속 → 별명 입력 → 캐릭터 고르기(왕·왕비 캐릭터가 인기) → "바닥 클릭으로 걷기"만 연습. 성공한 분마다 호명하며 박수 |
| 20–35분 | 자유 산책 + 갤러리 | 미로 대신 **갤러리방 관람**: 1살부터 100살까지 초상 9점 — "이 그림 전부 AI가 그렸습니다. 여러분 사진으로도 만들 수 있어요"가 최고의 정서적 훅 |
| 35–45분 | 휴식 | 기기에서 눈 떼기 |
| 45–65분 | 착석 강의 | 강의실 착석(E키는 보조 강사가 도움). ① AI는 이미 여러분 곁에 있다(내비, 스피커, 유튜브 추천) ② AI에게 말로 부탁하면 글·그림이 나온다 ③ **AI 안전**: 자녀 목소리 보이스피싱, 가짜 영상(딥페이크) 구별법 — "영상통화로 재확인", "돈 얘기는 무조건 끊고 직접 전화" |
| 65–85분 | AI 대화 체험 | 강사 화면(빔프로젝터)으로 AI와 대화 시연 → 수강생이 부르는 질문을 강사가 대신 입력 ("손주에게 보낼 생신 답장 써 줘" 등 생활 밀착형) → 여유 되는 분은 각자 기기로 |
| 85–90분 | 마무리 | 라운지에서 다 같이 이모지(❤️) 띄우기 기념 스크린샷, "오늘 배운 것 한 가지" 나눔 |

**60분 변형**: 갤러리 산책 10분, 휴식 생략, 대화 체험 15분.
**연속 강좌(4주) 확장**: 1주 입장·산책 / 2주 AI와 대화 / 3주 AI 사진 만들기 / 4주 AI 안전 — 매주 같은 공간에 "다시 들어오는 것" 자체가 복습이 된다.

## 진행 스크립트 요지
- 도입: "오늘은 외우실 것이 하나도 없습니다. 실수해도 고장 나지 않습니다"를 첫 마디로
- 갤러리에서: "100살 얼굴을 AI가 상상해서 그렸어요. 무섭게 쓰면 가짜 뉴스가 되고, 좋게 쓰면 추억이 됩니다" — 안전 교육으로 자연스럽게 연결되는 다리
- AI 안전은 공포가 아니라 요령으로: "3초 규칙 — 돈·개인정보 얘기가 나오면 3초 멈추고 끊는다"
- 채팅 대신 말로: 수강생 발화를 강사가 채팅으로 대신 입력하면 TTS가 읽어 줘서 "내 말이 저 공간에 울리는" 체험이 된다

## 준비물

- [ ] 큰 글씨 조작 안내지 1장 (그림 위주: ①바닥 클릭=이동 ②Esc=멈춤 ③🔊=소리)
- [ ] AI 안전 사례 자료 (보이스피싱 실제 사례 음성 등 — 공포 조장 않는 수위로)

## 평가/마무리
- 평가 없음. "오늘 성공한 것" 체크리스트(입장했다/걸었다/그림을 봤다/AI와 대화했다)에 스스로 동그라미 — 효능감이 산출물
- 종이 안내지에 사이트 주소 QR 인쇄 — "집에서 자녀·손주와 다시 들어와 보세요" (재방문이 최고의 복습)
