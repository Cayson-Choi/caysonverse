# caysonverse AI 수업 프로그램

caysonverse(웹 3D 메타버스)를 활용한 대상별 AI 교육 프로그램 기획 모음입니다.
초등학생부터 시니어까지 5개 대상에 맞춘 수업안과, 모든 수업에 공통으로 적용되는
운영 가이드·기능 로드맵으로 구성됩니다.

## 이 사이트가 수업에서 하는 3가지 역할

1. **가상 강의장** — 학생들이 접속해 모이는 수업 공간. 회원가입·설치가 전혀 없는
   "링크 클릭 → 닉네임 입력 → 입장" 구조라서 디지털 조작이 서툰 초등학생·시니어도
   1분 안에 들어올 수 있다. 강의실 20석 착석, 관리자 공지 배너, 채팅·이모지가
   수업 도구가 된다.
2. **AI가 만든 살아있는 증거물** — 이 세계 전체(3D 월드, 서버, 미로, 캐릭터)가
   AI(Claude)와의 대화만으로 만들어졌다. "AI가 뭘 할 수 있나요?"라는 질문에
   슬라이드 대신 **지금 서 있는 이 공간 자체**가 답이 된다. 갤러리방의
   AI 생성 초상화 9점(1살~100살)도 같은 증거물이다.
3. **바이브코딩 라이브 데모 무대** — 수업 중 강사가 AI에게 실시간으로 기능을
   시키고, 수강생들이 새로고침하면 눈앞에서 세계가 바뀐다. "말로 시키면
   소프트웨어가 만들어진다"를 체험시키는 가장 강력한 장치.

대상마다 이 3가지 역할의 비중을 다르게 섞는다. (초등·시니어는 ①② 중심,
중고등·대학생·직장인은 ③ 비중을 높인다.)

## 공통 수업 뼈대 (90분 기준)

모든 대상이 같은 구조를 쓰고, 각 단계의 내용물만 대상에 맞게 교체한다.

| 단계 | 시간 | 내용 | 사용하는 사이트 기능 |
|---|---|---|---|
| ① 접속 온보딩 | 10분 | 접속, 닉네임·캐릭터 선택, 조작 연습 | 게스트 입장, 캐릭터 8종, 라운지 |
| ② 아이스브레이킹 | 15분 | 미로 탈출 레이스 | 미로방, 탈출 전체 브로드캐스트 |
| ③ 착석 강의 | 25분 | 대상별 핵심 이론/스토리 | 강의실 20석, 공지 배너, 스크린 |
| ④ 체험/실습 | 30분 | 대상별 AI 실습 활동 | 채팅, 이모지, (외부 AI 도구 병행) |
| ⑤ 마무리 | 10분 | 소감 나눔, 이모지 반응, 갤러리 관람 | 라운지, 갤러리방, 이모지 6종 |

60분 수업은 ②를 10분·④를 20분으로 압축, 120분 수업은 ④를 두 세션으로 나눈다.
각 대상별 문서에 변형 타임테이블이 있다.

## 규모 제약 (기획 시 항상 지킬 것)

- 서버 동시 접속 정원: **단일 룸 100명** (기술 정원 110 = 100 + 재접속 여유 10)
- 강의실 착석: **20석** (4행×5열) — 20명 초과 회차는 착석을 조별 교대로 운영
- 상태 저장 없음(인메모리) — 서버 재배포/크래시 시 전원 퇴장 + 월드 초기화.
  수업 직전·수업 중 배포 금지.
