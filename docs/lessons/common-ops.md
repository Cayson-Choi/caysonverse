# 공통 운영 가이드

모든 대상·모든 회차에 공통으로 적용되는 수업 운영 방법입니다.
여기 적힌 기능은 전부 현재 사이트에 실제 구현되어 있는 것만 다룹니다.

## 1. 수업 전 준비

### 준비물 체크리스트
- [ ] 접속 URL 확정 + **QR 코드 인쇄/슬라이드 삽입** (모바일 접속 대비)
- [ ] 관리자 코드(`ADMIN_CODE` 환경변수) 확인 — 미설정이면 공지/강퇴 기능이 꺼져 있음
- [ ] 수업 24시간 전부터 서버 재배포 금지 (인메모리 상태 — 배포 시 전원 퇴장)
- [ ] 강사 PC 2개 브라우저 준비: ① 강사 아바타용 ② 관리자/예비용
- [ ] 예상 인원 확인 — 정원 100명, 초과분은 입장 화면에서 "정원이 가득 찼습니다" 안내됨
- [ ] (라이브 데모 회차) 강사 PC에 개발 환경 + `npm run dev` 동작 확인

### 접속 안내 문구 (그대로 사용 가능)
> 1. 카메라로 QR을 찍거나 주소를 입력해 접속합니다.
> 2. 닉네임(2~12자)을 입력하고 마음에 드는 캐릭터를 고릅니다.
> 3. [입장]을 누르면 끝! 가입도 설치도 없습니다.

## 2. 조작법 안내 스크립트

접속 직후 라운지에서 3분간 다 같이 연습시킨다.

**PC:**
- 이동: `W A S D` 또는 방향키, **또는 바닥을 클릭**하면 그 자리로 걸어감 (Esc로 정지)
- 시점: 마우스 드래그로 회전, 휠로 줌 (끝까지 당기면 1인칭), `V` 1인칭 전환, `M` 지도 보기
- 채팅: `Enter` → 입력 → `Enter` (머리 위 말풍선 6초)
- 이모지: 숫자키 `1`~`6` (👍 ❤️ 😂 👏 🎉 🙋)
- 앉기: 강의실 의자 근처에서 `E`

**모바일/태블릿:**
- 이동: 좌하단 조이스틱, 또는 바닥 터치
- 시점: 한 손가락 드래그 회전, 두 손가락 핀치 줌, 👁 버튼(1인칭), 🗺 버튼(지도)
- 채팅·이모지·앉기: 화면 버튼

**연습 순서(스크립트):** "다 같이 걸어보세요 → 바닥을 클릭해서 이동해 보세요 →
숫자 1을 눌러 엄지척을 띄워 보세요 → 채팅으로 자기 이름을 쳐 보세요."

## 3. 관리자(강사) 기능 사용법

- **로그인**: 입장 화면에서 관리자 코드 입력 (IP당 분당 5회 제한 — 오타 주의)
- **전체 공지 배너**: 화면 상단 고정 배너. 늦게 입장한 사람에게도 보이므로
  "지금 강의실로 모이세요" 같은 단계 전환 안내에 사용
- **강퇴**: 수업 방해 시 사용. 익명 입장이므로 재입장 자체는 막을 수 없음을 유의

## 4. 수업 중 단계 전환 운영

각 단계 전환은 **공지 배너 + 채팅 + (선착순 게임화)** 3중으로 안내한다.

- 라운지 → 미로: "미로방(🌀 포스터)으로 모이세요. 탈출하면 전체 방송이 나갑니다!"
  - 탈출 시 "🎉 ○○님이 미로를 탈출했습니다!" 전체 브로드캐스트가 뜨므로
    순위 경쟁이 자연스럽게 생긴다 (같은 사람 30초 쿨다운)
  - 미로 중앙 골에 라운지 귀환 포탈이 있어 탈출자는 자동 복귀 동선
- 라운지 → 강의실: "📚 포스터 방으로 이동, 의자 근처에서 E(또는 버튼)로 착석"
  - 좌석은 선착순 20석, 1의자 1인 (서버가 보장). 초과 인원은 뒤쪽 스탠딩
  - 착석 상태에서도 채팅·이모지 가능 — 강의 중 리액션 도구로 활용
- 갤러리 관람: "🖼 포스터 방 — 최무호 일대기. 이 그림들 전부 AI가 그렸습니다"

## 5. 발표/화면 공유 (현재 미구현 — 우회 방법)

사이트 안 화면 공유·음성은 아직 없다 (설계만 보존됨, design.md 15장).
- **오프라인 강의장**: 빔프로젝터에 강사 화면을 띄우고, 수강생은 사이트 속에서 착석
- **온라인 원격**: Zoom 등 화상도구로 화면·음성, 사이트는 "몸이 있는 공간"으로 병행
- 사이트 내 텍스트 전달은 공지 배너 + 채팅으로 충분

## 6. 트러블슈팅

| 증상 | 대응 |
|---|---|
| "정원이 가득 찼습니다" | 정원 100명 초과. 유휴 접속(중복 탭) 정리 요청 |
| 접속이 잠깐 끊김 | 20초 내 같은 위치·캐릭터로 자동 재접속됨. 기다리라고 안내 |
| 다른 사람 아바타가 반투명 | 그 사람이 순단 상태라는 표시. 곧 복구되거나 사라짐 |
| 채팅이 안 보내짐 | 200자 제한 + 5초당 3회 제한. 잠시 후 재시도 |
| 채팅이 소리로 들림/시끄러움 | 근처 사람 채팅 TTS 낭독 기능. 🔊/🔇 버튼으로 개인별 음소거 |
| 키보드로 안 움직여짐 | 채팅 입력창에 포커스가 있는 상태. Esc로 빠져나오기 |
| 3D가 느림 | 다른 탭·앱 종료, 창 크기 축소. 최후엔 새로고침 |
